## 联署机制 / Co-signing mechanism

联署通过 Pull Request 完成。任何人——个人、组织、研究者、从业者——都可以通过 PR 把自己加入对应公开信的 `signatories.md`。

Co-signing is done via Pull Request. Anyone — individual, organization, researcher, practitioner — can add themselves to the corresponding `signatories.md` via PR.

具体步骤见 [CONTRIBUTING.md](../CONTRIBUTING.md) 第三节"翻译/议题报告/代码/资金支持/传播"中的提案模板说明，以及每封信内 `signatories.md` 的格式约定。
