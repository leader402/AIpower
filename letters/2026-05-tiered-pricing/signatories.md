## 联署须知 / Notes for co-signers

1. **真实意愿**：请确认你已阅读 [letter.md](letter.md) 全文，认同其全部诉求，再签署。
2. **身份可选**：你可以选择不公开自己的雇主或行业身份。仅 GitHub ID 与国家/地区为必填。
3. **匿名联署**：原则上要求 GitHub 实名账号，但若你有正当的安全顾虑（举报人、记者、敏感地区从业者），可通过 [pledges/anon](../../pledges/) 目录提交匿名签名，由秘书处审核后以 *"Anonymous-XXX, [country/region]"* 形式入表。
4. **机构联署**：请确保你有代表该机构签署的内部授权；秘书处可能在合并 PR 前要求出示证明（不公开）。
5. **撤销**：任何联署人均可通过 PR 删除自己。无需说明理由。
